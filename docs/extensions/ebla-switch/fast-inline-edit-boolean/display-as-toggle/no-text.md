# Ebla Switch. Display As Toggle. No Text

This feature allows you to customize the text of the toggle when the value is false.

## How to use it

1. go to **Admin** -> **Entity Manager** -> **Scope** -> **Fields** -> **Add Field** -> **Boolean**.
2. Enable **Display As Toggle**.
3. Select **No** in the **No Text** option.
