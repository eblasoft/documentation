# [Ebla IFrame.](../setting-up.md) Iframe Field

 **this feature enables you to add a field IFrame .**

### How to use it

1. go to **Admin** -> **Entity Manager** -> **Scope** -> **Fields** -> **Add Field** -> **IFrame**.

![IFrame Field](../../../_static/images/extensions/ebla-iframe/iframe-field/iframe-field.png)


![IFrame Field](../../../_static/images/extensions/ebla-iframe/iframe-field/iframe-field-1.png)

2. Add your URL in the field.

![IFrame Field](../../../_static/images/extensions/ebla-iframe/iframe-field/iframe-field-2.png)
