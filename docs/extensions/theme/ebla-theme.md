## Features

* New theme (**Eblasoft**)
* Add login Background Image.
* Add Favicon.
* Logo Position : center - left - right.
* Open Side Menu Hover.
* Ability to change Colors:
    - Top navbar color
    - Tabs menu background color
    - Logo background (side).
    - Text Color.
    - Brand Info.
    - Brand Danger.
    - Background Color.
    - Brand Success
    - Brand Warning
    - Brand Primary

go to **Administration** -> **Ebla Theme**
