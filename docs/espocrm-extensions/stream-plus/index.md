# Key Features
### Custom Stream Fields

* Add new fields to Note entities directly from Entity Manager.
* New fields appear automatically in the Stream for all users.
* Supports any standard field type available in EspoCRM, enhancing customization.
* Improves data visibility and workflow by allowing tailored information to be captured per record.

![Stream Plus](../../_static/images/espocrm-extensions/stream-plus/stream-plus.png)

---

### Access & Configuration

**-** Go to **Administration** -> **Entity Manager** -> **Note.**

Configure new fields as usual; they will instantly reflect in the Stream.



![Stream Plus](../../_static/images/espocrm-extensions/stream-plus/stream-plus-op.png)


!!! note

     Ability to create reports on Stream note entity using Advanced pack.
