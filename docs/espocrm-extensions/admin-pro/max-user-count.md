# Max User Count

> Max User Count is a feature in EspoCRM
> of [Ebla Admin Pro](https://www.eblasoft.com.tr/espocrm-extension-page/espocrm-admin-pro) extension.

___


**-** Go to **Administration > Users** .

![max-user-count](../../_static/images/espocrm-extensions/admin-pro/max-user-count.png)

!!! note

     If the value is set to 0 or left empty, no limitation will be applied.
