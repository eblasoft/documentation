# AI Video Generation

Video Generation is not implemented in the current extension build.

## Current Status

At the time of this documentation update, the installed extension does not include:

- A video-generation UI
- A video-generation backend route
- A registered formula function for video generation

## What This Means

If you are using the current build, do not expect:

- A **Generate Video** button in record fields
- A video-generation modal
- `eblaAi\generateVideo(...)` in formula

## Recommended Alternatives

For now, use:

- [Image Generation](image-generation.md) for visual asset creation
- [Voice Generation](voice-generation.md) for speech generation
- [Formula](formula.md) for currently supported automation functions
